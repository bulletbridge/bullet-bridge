# Trademark And Branding

Bullet Bridge is an independent project.

## Bullet Bridge Branding

The `Bullet Bridge` name, `BB` icon, project artwork, screenshots, and related
visual identity belong to the Bullet Bridge project.

Do not use the Bullet Bridge name, icon, or confusingly similar branding for
forks, modified builds, browser extension listings, websites, packages, or
services without prior written permission.

If permission is granted for a fork or derivative work, it must use a clearly
different name and visual identity unless the Bullet Bridge project explicitly
allows otherwise in writing.

## Pushbullet Compatibility

Pushbullet is a trademark of its respective owner.

The Pushbullet name is used only to describe compatibility with the Pushbullet
service. Bullet Bridge is not affiliated with, endorsed by, sponsored by, or
provided by Pushbullet.

Do not use Pushbullet's logos, icons, product artwork, or official branding in
Bullet Bridge forks, screenshots, extension listings, or promotional material
unless you have permission from the Pushbullet trademark owner.

Acceptable wording:

- `Bullet Bridge is an unofficial client for Pushbullet users.`
- `Works with Pushbullet through the Pushbullet API.`
- `Independent extension for Pushbullet users.`

Avoid wording that implies endorsement or official status:

- `Official Pushbullet extension`
- `Pushbullet for Chrome`
- `Pushbullet Bridge`
- `Approved by Pushbullet`
- `Partnered with Pushbullet`
