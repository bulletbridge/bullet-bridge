## Name

Bullet Bridge

## Summary

Send, receive, search, and manage Pushbullet pushes from a modern browser.

## Category

Productivity

## Language

English

## Detailed Description

Bullet Bridge is an independent browser extension for Pushbullet users. It
brings Pushbullet sending, receiving, file sharing, mirrored notifications, push
history, media previews, search, and right-click sharing back to modern
Chromium-based browsers.

Bullet Bridge is not affiliated with, endorsed by, sponsored by, or provided by
Pushbullet.

Features:

- Sign in with Pushbullet OAuth.
- Send notes, links, current tabs, and files.
- Send to all devices or a selected device.
- Browse recent push history in a compact chat-style popup.
- Load older pushes on demand.
- Search loaded push history by message, URL, file name, and device.
- Preview image, video, file, and link pushes inline.
- Copy exact push content from chat bubbles.
- Delete individual pushes from history.
- Push pages, links, selected text, and image URLs from the right-click menu.
- Receive pushes through Pushbullet realtime events.
- Show received pushes as browser notifications.
- Show mirrored Android notifications in the popup and as browser notifications.
- Dismiss mirrored Android notifications locally.
- Open pushed links from browser notifications.

Privacy posture:

- No analytics.
- No ads.
- No trackers.
- No telemetry.
- No remote code.
- No third-party reporting services.
- Push data is sent only to Pushbullet and Pushbullet-provided upload endpoints.
